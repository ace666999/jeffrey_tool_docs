## 依赖版本
兼容所有 node 版本，推荐使用 pnpm 进行安装，也可直接使用 bun 工具进行快速安装。

## 安装

::: code-tabs

@tab npm
```
npm install jeffrey_tool
```

@tab pnpm
```
pnpm install jeffrey_tool
```

@tab bun
```
bun install jeffrey_tool
```
:::

## 引入使用

````javascript
import { mapTree } from "jeffrey_tool"
````
